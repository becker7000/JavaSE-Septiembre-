
### Ejercicio de variables:

Nota: todos los programas que podemos
crear tienen 3 instrucciones escenciales:

1. Entradas/salidas
2. Cálculos
3. Control
---

#### *Entradas*:
```
    a = 10
    b = 20
    c = 5
```

#### *Cálculos:*

```
    a = a + 3 = 10 + 3 = 13
    b = b + 4 - a = 20 + 4 - 13 = 11
    c = a + b + c = 13 + 11 + 5 = 29
    a = a + c = 13 + 29 = 42
    b = 4
    c = c + 3 - b + 2 = 29 + 3 - 4 + 2 = 30 
```

#### *Salidas:*
```
    Valores finales:
    a = 42, b = 4 y c = 30
```
